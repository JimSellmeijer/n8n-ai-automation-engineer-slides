# Bonus Task

**Themed Nicknames**: Extend the prompt to generate nicknames based on a theme (e.g., "space-themed", "superhero-themed", "medieval-themed").

Example Prompt Addition:
```
Theme: Space-themed
Create a space or astronomy related nickname.
```

**Multiple Variations**: Generate 3 different nickname options per name and let the user choose.

**Personality Integration**: Add a personality field (e.g., "funny", "serious", "creative") and generate nicknames based on the personality.
